# Проект обновления личного кабинета клиента в банке

## Описание:

Проект - это веб-приложение на Python для управления задачами и проектами.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Mary19072025/homework.git
```
2. Установите зависимости:
```
pip install -r pythonproject-py3.13
```
## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Создайте новый проект и начните добавлять задачи.
3. Назначайте сроки выполнения и приоритеты для задач, чтобы эффективно управлять проектами.
